# Status: pending
# Dependencies: 6, 12, 13
# Priority: high
# Description: Google AdMob SDK를 통합하여 앱 내 배너 광고, 전면 광고를 구현하고 광고 제거 기능을 포함한 수익 모델을 구축합니다.
# Details:
1. Google AdMob SDK 통합:
   - google_mobile_ads 패키지 추가 (^3.1.0 이상)
   - AndroidManifest.xml 및 Info.plist 설정
   - 테스트 광고 ID 설정 (개발 단계)

2. 광고 서비스 구현 (services/ad_service.dart):
   - 배너 광고 초기화 및 로드
   - 전면 광고 초기화 및 로드
   - 광고 상태 관리
   - 광고 이벤트 처리 (클릭, 노출, 오류)

3. 배너 광고 구현:
   - 앱 하단 고정 배너 위젯 (widgets/banner_ad_widget.dart)
   - 광고 로딩 상태 처리
   - 광고 실패 시 대체 UI
   - 반응형 배너 크기 조정

4. 전면 광고 구현:
   - 주요 화면 전환 시 표시 (일기 저장 후, 앱 시작 시)
   - 광고 표시 빈도 조절 (사용자 경험 고려)
   - 광고 로딩 인디케이터
   - 광고 표시 전 카운트다운 (선택적)

5. 광고 제거 프리미엄 기능:
   - 인앱 결제와 연동 (Task 12 활용)
   - 구독 상태에 따른 광고 표시/숨김 처리
   - 광고 제거 혜택 안내 UI

6. 광고 수익 최적화:
   - 광고 표시 전략 (타이밍, 빈도)
   - A/B 테스트 프레임워크 연동
   - 사용자 세그먼트별 광고 전략
   - 광고 성과 분석 (Firebase Analytics 연동)

7. AdMob 계정 설정:
   - 실제 광고 단위 ID 생성 및 관리
   - 광고 필터링 설정 (부적절한 광고 차단)
   - 수익 지급 정보 설정
   - 개인정보 보호정책 연동

8. 광고 관련 사용자 설정:
   - 광고 개인화 동의 관리
   - 광고 표시 빈도 설정 (사용자 선택)
   - 광고 유형 선호도 설정 (선택적)

1. 테스트 광고 ID를 사용한 광고 로드 테스트:

   - 전면 광고 로드 및 표시 확인

   - 광고 로드 실패 시 오류 처리 확인
   - 광고 노출 이벤트 로깅 확인

3. 프리미엄 기능 연동 테스트:
