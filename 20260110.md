# 2026-01-10 개발 일지

## v1.1.0 릴리즈

### 주요 변경사항

#### 1. 사용자 설정 개선
- **사용자 이름 위치 변경**: 보안 관리 → 앱 설정 상단으로 이동
- **성별 선택 기능 추가**: 남성/여성/선택 안함 옵션
  - 초기 설정 화면에서 선택 가능
  - 설정 화면에서 변경 가능
  - AI 이미지 생성 시 프롬프트에 반영

#### 2. 새로운 썸네일 스타일 추가
- **색연필 스타일 (Color Pencil)**: 동화책 느낌의 색연필 일러스트
  - 프롬프트: Hand-drawn colored pencil illustration with clean line art, soft watercolor wash textures, bright pastel colors, storybook illustration feel

#### 3. 기존 스타일 개선
- **피규어 스타일**: 핑크톤 제거, 다양한 색상으로 변경
  - 기존: "soft pink and white colors"
  - 변경: "vibrant and varied colors"

#### 4. UI/UX 개선
- **썸네일 스타일 선택기**:
  - 체크 아이콘 제거, 테두리만 유지
  - 스타일 선택 시 스크롤 위치 유지 (StatefulWidget + ScrollController + PageStorageKey)
- **새 스타일 안내 팝업**: 색연필 스타일로 변경 (v2 팝업)

#### 5. 버그 수정
- **일기 작성 페이지 스타일 버튼**: 어린이그림/피규어/색연필 스타일 아이콘 누락 수정
- 기타 마이너 버그 수정
